# 前端页面使用说明

## 🚀 快速开始

### 方式一：直接打开（使用模拟数据）

直接用浏览器打开以下页面即可，**无需启动 Mock 服务**：

- **登录页**: `login.html` ⭐ **推荐入口**
- **学生订餐页**: `student-order.html`
- **管理员统计页**: `admin-stats.html`

> ✅ 页面已内置模拟数据，即使没有 Mock 服务也能完整演示！

### 方式二：使用快速启动脚本

双击运行：
```
test-pages.bat
```

可以选择打开登录页、学生页、管理员页或全部打开。

### 方式三：启动 Mock 服务（使用 API 契约数据）

```bash
cd ../mock
docker-compose up -d
```

然后刷新页面，页面将自动从 Mock API 获取数据。

## 🔧 技术细节

### 模拟数据机制

1. **自动降级**: 页面会先尝试调用 Mock API
2. **超时检测**: 3秒未响应则自动使用模拟数据
3. **友好提示**: 显示黄色提示条，3秒后自动消失
4. **完整功能**: 模拟数据支持所有交互操作

### 数据流向

```
页面加载 → 尝试 Mock API (3s)
           ├─ 成功 → 使用 API 数据
           └─ 失败 → 使用内置模拟数据
```

## 📸 演示步骤

### 完整登录流程演示

1. 打开 `login.html`
2. 选择角色（学生/管理员）
3. 点击快速测试账号
4. 自动登录并跳转到对应页面

### 学生订餐演示

1. 从登录页点击"张三"测试账号
2. 自动跳转到学生订餐页
2. 查看菜品列表（6个菜品）
3. 点击"整份"或"半份"按钮添加到购物车
4. 在右侧购物车查看已选菜品
5. 点击"提交订单"
6. 查看订单成功弹窗
7. 点击右上角"退出登录"返回登录页

### 管理员统计演示

1. 从登录页选择"管理员"角色
2. 点击"管理员"测试账号
3. 自动跳转到管理员统计页
4. 查看概览卡片（订单数/人数/菜品）
5. 查看柱状图（菜品需求量）
6. 查看饼图（份量分布）
7. 查看数据表格
8. 打开浏览器控制台，运行 `runBoundaryTests()` 查看边界测试

## ⚠️ 注意事项

1. **无需后端**: 页面完全独立运行
2. **浏览器要求**: 支持现代浏览器（Chrome/Edge/Firefox）
3. **网络要求**: 需要联网加载 Bootstrap 和 Chart.js CDN
4. **控制台**: 按 F12 可查看日志和调试

## 🐛 故障排查

| 问题 | 解决方案 |
|------|---------|
| 页面空白 | 检查浏览器控制台错误 |
| 样式丢失 | 检查网络连接（需加载 CDN） |
| 图表不显示 | 确保 Chart.js 加载成功 |
| 数据不显示 | 查看控制台日志 |
